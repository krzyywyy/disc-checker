# Disc Checker 1.4.3 Release Notes

## Highlights

- Further reworked background execution to avoid visible CrystalDiskInfo detection window.
- UAC prompt behavior remains unchanged.

## What changed in 1.4.3

- Switched hidden-run mechanism to an elevated temporary Task Scheduler job.
- CrystalDiskInfo is triggered in background context from that elevated task.
- Updated package metadata and docs to 1.4.3.

## Validation scope

This version was tested on **only two computers**.
