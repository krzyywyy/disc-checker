Disc Checker

NVDA add-on for checking physical disk health using bundled CrystalDiskInfo.

How to run

  • NVDA menu > Tools > Check disk health
  • Keyboard shortcut: NVDA+Shift+D

Report content

  • Disk health percentage
  • Disk temperature
  • All available disk key/value fields parsed from CrystalDiskInfo report sections

Behavior

  • The check always runs in the background.
  • Each check requires UAC elevation.
  • If UAC is denied, the check is aborted.
  • No separate software installation is required by the end user.

Validation note

This add-on has been tested on only two computers.
