# Disc Checker 1.4.2 Release Notes

## Highlights

- CrystalDiskInfo no longer shows its own visible check window.
- Disk check now runs hidden in the background.
- UAC prompt behavior is unchanged (still shown every run).

## What changed in 1.4.2

- Reworked CrystalDiskInfo launch flow to use hidden background execution.
- Kept elevation flow intact via UAC.
- Updated package metadata and docs to 1.4.2.

This version was tested on **only two computers**.
